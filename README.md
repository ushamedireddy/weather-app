# 🌤️ Weather App

A responsive weather app built with HTML, CSS and JavaScript 
that shows real-time weather data for any city in the world.

## ✨ Features
- 🔍 Search weather by any city
- 📍 Get weather by current location
- 🌡️ Toggle between Celsius and Fahrenheit
- 💧 Humidity, Wind Speed, Feels Like
- 🕐 Recent search history
- 🎨 Dynamic background based on weather
- 📱 Fully responsive on all devices

## 🚀 How to Run Locally
1. Clone the repository
   git clone https://github.com/ushamedireddy/weather-app.git
2. Open index.html in your browser
3. Search for any city!

## 📡 API Used
- [Open-Meteo API](https://open-meteo.com/) - Free weather API
- [Geocoding API](https://geocoding-api.open-meteo.com/) - City search
